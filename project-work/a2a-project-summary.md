# A2A Receptor MD Project Summary

The final project focused on the human adenosine A2A receptor and asked whether AlphaFold2 MSA-depth ensembles, combined with normal-mode analysis, can recover part of the conformational subspace sampled by molecular dynamics.

This GitHub version uses the final A2A MD project materials, not the earlier project pitch deck.

## Core idea

AlphaFold2 can predict highly accurate static structures, but GPCR function depends on motion. Molecular dynamics gives richer dynamic information but is expensive. The project tested whether shallow/intermediate MSA-depth AF2 ensembles plus NMA could cheaply approximate meaningful MD-like motions.
