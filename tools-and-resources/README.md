# Tools and Resources

This folder summarizes the `CompBio26 resources hub` by use case.

## Main categories

- Molecular dynamics preparation and analysis.
- OpenMM and AMBER/CPPTRAJ workflows.
- AlphaFold, ColabFold, AFDB, Foldseek, FoldMason, and structure prediction tools.
- Protein databases and annotation platforms such as PDBe, InterPro, HMMER, STRING, and Anvi'o.
- AI/ML explainers for transformers, CNNs, GNNs, and neural networks.
- Quantum computing resources and Kaiwu/QUBO-related tools.

See the full categorized discussion in [`../WORKSHOP_SUMMARY.md`](../WORKSHOP_SUMMARY.md#21-resource-hub).
